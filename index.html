クラス統計 (Class Statistics)

指定した学年・テスト時点でのクラス全員の成績一覧表を作成します。
「統計・グラフ1」の平均点および「統計・グラフ2」の累積評価（単純・加重・GPA）とそれぞれの順位を表示します。

・59点以下が4科目以上 / ・49点以下が3科目以上 / ・39点以下が2科目以上
【通年平均ベース】 以下のいずれかに該当する学生を抽出：
・平均59点以下が3科目以上 / ・平均49点以下が2科目以上 / ・平均39点以下が1科目以上
条件を選択して「抽出実行」をクリックしてください。

基本的な使用方法 (Basic Usage)

  1. 1. 初期設定（学生・科目の登録）:
    「各種設定 (Settings)」タブで、クラスの学生名リストと、各学年の科目定義（単位数や種類）を整えます。CSVファイルからのインポートも可能です。
  2. 2. 成績の入力:
    「成績一覧 (Grades)」タブで、学生と学年を選択して点数を入力します。
    ★効率化テクニック: 表の見出しにあるアイコンをクリックすると、Excel等からコピーした1列分の成績をまとめて貼り付けできます。
  3. 3. 個人の分析:
    「統計・グラフ」タブで、偏差値の推移、順位の変動、学年ごとの評点平均(GPA)などを確認できます。
  4. 4. クラス全体の集計:
    「クラス統計」タブでは、特定のテスト時点におけるクラス全員の成績順位表を作成し、CSVやPDFで出力できます。
  5. 5. データの保存:
    入力したデータは随時ブラウザに保存されますが、バックアップとして定期的に「JSON」ボタンからファイルを保存することを推奨します。

セキュリティポリシー (Security Policy)

本システムは、学生の個人情報および成績データを安全に取り扱うため、以下の設計指針を遵守しています。

  • ネットワーク送信の有無: 入力された学生名、科目、成績等のデータがインターネット上のサーバーに送信されることは一切ありません。すべての処理はブラウザ内で完結します。
  • データの保存場所: データは、現在使用しているブラウザのLocalStorage（ローカルストレージ）にのみ保存されます。ブラウザやPCを跨いでの同期機能もありません。
  • 共有について: システムが自動的に他者とデータを共有することはありません。共有が必要な場合は、明示的に書き出したファイルを送受する必要があります。
  • 利用端末の制限: 第三者がアクセス可能な共用端末での使用は避け、必ず教員本人しかアクセスしない保護された端末で使用してください。

生成可能ファイルと保存内容

ファイル形式 主な用途 含まれる主要内容
PDF (印刷) 成績レポート、帳票出力 個人名、各テスト点数、平均・順位推移グラフ、分布図
JSON 全データの完全バックアップ 学生リスト、科目定義、入力済みの全スコアデータ
CSV (書き出し) Excel等での分析・加工 学生一覧、科目一覧、クラス成績統計表(点数・順位)

重要・免責事項

本システムで算出・表示されるデータは、あくまで学習指導の参考とするための「参考値」です。

平均点、順位、GPA、累計単位数等の計算結果には、システム上の端数処理や各種推定値が含まれる場合があります。単位修得の可否や進級判定等の最終的かつ公式な判断は、必ず教務主事室が発行する正式な書類（通知表、成績証明書等）を正として判断してください。
